# Tesis - Maestría en Ciencia de Datos

Repositorio de trabajo para la tesis sobre predicción de ingresos utilizando la Encuesta Continua de Hogares (ECH) y aprendizaje automático.

ECH 2024 es el notebook que contiene el EDA y el Modelado inicial con los modelos simples funcionando sin sobreajustar.

ECH 2024 EDA + ECH 2024 Modelado (ECH 2024 partido en 2) son el EDA y el Modelado donde el preprocesamiento con las transformaciones se hace basicamente en el Modelado una vez realizado el Split.

ECH 2024 EDA 2 + ECH 2024 Modelado 2 son el EDA y el Modelado donde el preprocesamiento con las transformaciones se hace todo en el EDA. El Modelado aranca haciendo el Split con el dataset depurado.
